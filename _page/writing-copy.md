Most editing is done inside text boxes.
## Copying and Pasting

Sometimes, copying and pasting text from Word, Emails, or from PDFs will cause problems with the website. This is because copy and pasting also copies some of the code from these documents, as well as the written text.

After copying and pasting 

Clear formatting

Paste into Text view

Do not copy and paste images into the text box.

Instead, click Insert > Image and upload the image.

## Spelling

Always check spelling before publishing, and if in doubt, check The Courtauld style guide.

Do not use the spell check on the website, because it only works with American spelling.
